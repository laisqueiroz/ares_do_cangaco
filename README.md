# Os Ares do Cangaço ✈️🌵

Uma experiência interativa em **Realidade Aumentada (RA)** que une a história de Quixadá ao universo do Cangaço. O projeto faz uma releitura artística baseada em dois fatos históricos da região: o pouso de um hidroavião comercial no Açude do Cedro em 1930 e a arma presenteada por Lampião ao Cego Aderaldo.

Desenvolvido como projeto acadêmico na disciplina de **Modelagem Tridimensional** na **UFC - Campus Quixadá**.

## 🚀 Como Acessar

O projeto está online e pode ser acessado diretamente pelo navegador do seu smartphone:

1. Acesse o site através do link do GitHub Pages:  
   `https://seu-usuario.github.io/os-ares-do-cangaco/`
2. Dê permissão para o site utilizar a câmera do seu dispositivo.
3. Aponte a câmera para o marcador **`marker.patt`** para visualizar o modelo 3D do hidroavião interativo.

## 🛠️ Tecnologias Utilizadas

* **HTML5 / CSS3** – Estrutura da Landing Page e Galeria de Mídia.
* **A-Frame** – Framework web para criação de experiências 3D e VR.
* **AR.js** – Tecnologia utilizada para renderizar a Realidade Aumentada via marcador no navegador.
* **GitHub Pages** – Hospedagem gratuita do projeto sob protocolo seguro (HTTPS).
